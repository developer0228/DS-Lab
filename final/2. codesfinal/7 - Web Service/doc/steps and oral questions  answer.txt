Steps to Perform Calculator Web Service in Java using VS Code 

Step 1: Open VS Code 

Install: 

Extension Pack for Java 

 

Step 2: Create Project Folder 

WebService 

Open the folder in VS Code. 

 

Step 3: Create Server File 

Create file: 

CalculatorServer.java 

Paste the Calculator Server Java code and save the file. 

 

Step 4: Create Client File 

Create file: 

CalculatorClient.java 

Paste the Calculator Client Java code and save the file. 

 

Step 5: Compile Programs 

Open terminal in VS Code: 



  
javac CalculatorServer.java 
javac CalculatorClient.java 

 

Step 6: Run the Server 

Open terminal 1: 

java CalculatorServer 

Output: 

Server Started 

 

Step 7: Run the Client 

Open terminal 2: 

java CalculatorClient 

 

Expected Output 

Server Side 

Addition = 30 

 

Client Side 

Addition = 30

A web service is a software component that allows communication between applications 
over a network.
