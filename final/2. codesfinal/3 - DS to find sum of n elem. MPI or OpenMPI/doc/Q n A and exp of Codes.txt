Practical 3: Develop Distributed System to Find Sum of N Elements using MPI/OpenMP
1. Aim

To develop a distributed system using MPI/OpenMP for calculating the sum of N array elements by distributing work among multiple processors.

2. Explanation of Practical
What is MPI/OpenMP?
MPI

MPI stands for Message Passing Interface.
It is used for communication between multiple processors in distributed systems.

OpenMP

OpenMP stands for Open Multi-Processing.
It is used for parallel programming using multiple threads in shared memory systems.

In this practical:

Array is divided among processors.
Each processor calculates partial sum.
Final sum is obtained by combining intermediate sums.

5. How it is Implemented
Important Steps
1. Divide Array
N elements are divided among n processors.

Example:
100 elements distributed among 4 processors.

2. Assign Work

Each processor receives subset of array.

Example:

P1 → 1–25
P2 → 26–50
3. Calculate Intermediate Sum

Every processor computes local sum independently.

4. Combine Results

All intermediate sums are added to obtain final sum.

5. Display Partial Sums

Each processor displays its calculated sum.

6. Working of Distributed Sum Calculation
Input Array
      ↓
Array Divided Among Processors
      ↓
Each Processor Calculates Partial Sum
      ↓
Intermediate Sums Generated
      ↓
Final Sum Computed
